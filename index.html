> web projects

I've created many websites for many purposes. Here are a select few.

Leafarctica

Originally forked from another project, this web map displays antarctica using a polar projection, supplemented by some open source marker and boundary data and MODIS imagery.

URL: http://ajzeigert.github.io/leafarctica/

Tarot Reader

Created in order to teach myself React.

URL: https://ajzeigert.github.io/tarot-reader/

Robomapper

A fun little bot that spits out maps of places

URL: https://twitter.com/robomapper

XYZ Export Tool

This app allows the user to upload a KMZ file. Any points in the file are displayed on the map. Elevation and coordinates in a local projection are determined. The user can then export a table with this information. Developed for internal use. Built using ArcGIS Javascript API.

URL: Sorry, not publicly available

Project Delivery

This app lets clients track the progress of their work, access documents, photos and contacts. Built using React, Node and Postgres. Hosted on AWS.

URL: Sorry, not publicly available

Safety Hub

This app allows employees at my company to report safety information. It is built using React and the Microsoft Graph API. It reads from and records data to a shared spreadsheet on Sharepoint Online, thus it does not require a backend server.

URL: Sorry, not publicly available
